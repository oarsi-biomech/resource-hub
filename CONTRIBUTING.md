# Contributing to the OARSI Biomechanics Resource Hub

Thank you for helping improve this community resource.

## What to contribute in v1

In v1, the main contribution type is a **new or updated lab/group/center entry** in `labs/`.

## Quick start (recommended)

1. Open `templates/lab-entry-template.md`.
2. Copy it into a new file in `labs/` (for example: `labs/example-lab-name.md`).
3. Fill required fields.
4. Remove any optional sections that are empty.
5. Add your new entry link to `labs/index.md`.
6. Open a pull request using the repository PR template.

## Entry quality checklist

Before submitting, confirm:

- [ ] The entry is relevant to OA biomechanics or closely related human movement research.
- [ ] All required fields are completed.
- [ ] Optional sections are removed if empty (no blank headings).
- [ ] Links are valid and clearly labeled.
- [ ] Content is respectful, professional, and clear.

## Scope guidance

### Include
- OA biomechanics labs
- Human movement labs/groups with clear OA relevance
- Related institutional centers that support OA biomechanics work

### Do not include (v1)
- Dataset hosting uploads
- Code hosting uploads
- Complex affiliation metadata or relationship automation

## For first-time GitHub contributors

If you are new to GitHub:

- You can use the **Edit** (pencil) button in the web interface.
- GitHub will help you fork the repository and open a pull request automatically.
- Use the pull request checklist to confirm your update.

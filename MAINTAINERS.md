# Maintainers

This file defines lightweight governance for v1 of the OARSI Biomechanics Resource Hub.

## Maintainer responsibilities

Maintainers should:

- Review pull requests for scope fit and clarity.
- Ensure required fields are present in entries.
- Ensure optional empty headings are removed.
- Keep `labs/index.md` synchronized with entry files.
- Encourage a welcoming contributor experience.

## Review expectations

For v1, reviewers should prioritize:

1. Relevance to OA biomechanics/human movement research.
2. Basic completeness and readability.
3. Working links and sensible contact details.
4. Simplicity over process overhead.

## Decision model

- Normal changes: one maintainer approval is sufficient.
- Structural changes (template, governance, folder layout): prefer two maintainer approvals.

# OARSI Biomechanics Resource Hub

A community-driven directory for osteoarthritis biomechanics labs, groups, and centers.

## Explore the directory

➡️ **[Browse labs, groups, and centers](labs/index.md)**

## About v1

Version 1 is intentionally simple and focused:

- A curated directory of OA biomechanics-relevant **labs, groups, and centers**.
- A lightweight contribution workflow that works for first-time GitHub contributors.
- A structure designed for GitHub Pages and future expansion.

## What may come in future versions

As the community grows, this hub may expand to include:

- Code resources
- Dataset/resource links
- Additional community knowledge pages

(These are not part of v1 yet.)

Want to contribute to this resource? See **[CONTRIBUTING.md](https://github.com/oarsi-biomech/resource-hub/blob/main/CONTRIBUTING.md)** for what we welcome and how to get started.
